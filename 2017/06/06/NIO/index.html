什么是NIO

NIO是New I/O的简称，与旧式的基于流的I/O方法相对，从名字看，它表示新的一套Java I/O标准。它是在Java 1.4中被纳入到JDK中的，并具有以下特性：

  • NIO是基于块（Block）的，它以块为基本单位处理数据
  • 为所有的原始类型提供（Buffer）缓存支持
  • 增加通道（Channel）对象，作为新的原始 I/O 抽象
  • 支持锁和内存映射文件的文件访问接口
  • 提供了基于Selector的异步网络I/O
